암기단어장 개인정보 취급방침

최종 수정일: 2025년 3월 19일

1. 개요

암기단어장은 사용자의 프라이버시를 최우선으로 고려하며, 개인정보 보호를 매우 중요하게 생각합니다. 본 앱은 원칙적으로 사용자의 민감한 개인정보를 수집하거나 외부 서버에 저장하지 않습니다.

2. 정보 수집 및 사용

암기단어장은 사용자가 구글 계정으로 로그인할 경우, 구글 OAuth 인증을 통해 다음과 같은 최소한의 범위에서 정보에 접근합니다:

  • 사용자의 Google Drive 내 Google 스프레드시트 목록
  • 사용자가 선택한 특정 Google 스프레드시트의 콘텐츠 (단어 목록)

이 데이터는 앱 내에서 단어 등록 기능을 제공하기 위한 용도로만 사용되며, 다음과 같은 조건을 따릅니다:

  • 어떠한 정보도 외부 서버에 저장, 전송, 공유하거나 제3자에게 판매하지 않습니다.
  • 구글 스프레드시트를 통해 가져온 데이터는 오직 사용자의 앱 로컬 기기에 저장되며, 앱 내 기능을 수행하는 데에만 사용됩니다.
  • 접근 권한은 사용자가 직접 허용한 범위에 한정되며, 앱은 사용자 동의 없이 추가 데이터를 요청하거나 활용하지 않습니다.

3. 어린이 프라이버시

암기단어장은 13세 미만의 아동으로부터 고의적으로 개인정보를 수집하지 않습니다. 만약 부모 또는 보호자가 13세 미만 아동의 정보가 수집되었음을 인지한 경우, 문의 주시면 지체 없이 삭제 조치를 하겠습니다.

4. 보안

암기단어장은 사용자 데이터를 외부로 전송하지 않으며, 모든 데이터는 사용자의 기기에만 저장됩니다. 따라서 데이터의 보안은 전적으로 사용자의 기기 보안 설정에 따라 보호됩니다. 앱은 데이터를 외부에 노출하지 않으며, 서버 기반 데이터 처리를 하지 않습니다.

5. 변경사항

개인정보 처리방침이 변경될 경우, 변경 내용을 이 페이지에 게시하며, 앱 내에서 별도로 안내합니다. 변경 사항은 게시 시점부터 즉시 적용됩니다.
