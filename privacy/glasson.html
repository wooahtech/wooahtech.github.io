안경엔 개인정보 취급방침

최종 수정일: 2025년 9월 13일

1. 개요

"안경엔"은 사용자의 프라이버시를 존중하며 개인정보 보호를 중요하게 여깁니다. 본 앱은 서비스 제공에 필요한 최소한의 개인정보만을 수집하며, 수집된 정보는 명시된 목적에 한해서만 사용됩니다.

2. 수집하는 개인정보 항목

- 소셜 로그인 선택 시 해당 플랫폼에서 제공받은 식별번호(예: Kakao/Google/Apple 고유 ID)를 수집합니다.
- 사용자가 직접 선택하여 업로드한 사진(안경 사진 및 얼굴 사진)을 수집합니다.
- 얼굴 특징(landmarks: 눈, 코, 입, 얼굴 윤곽선 등)의 추출 또는 분석은 본 앱 내에서는 직접 수행하지 않으며, 오직 외부 API(예: Google API)를 통해 렌더링된 가상 착용 이미지 생성 목적을 위해 처리됩니다.

3. 개인정보의 수집 및 이용 목적

- 소셜 로그인 식별번호: 회원 식별 및 로그인 인증.
- 업로드된 사진(안경 및 얼굴 사진): 가상 착용 이미지(virtual try-on) 제공을 위한 이미지 전달 및 외부 API 처리.
- 외부 API(예: Google API)를 통해서만 사진 처리(렌더링) 이루어지며, 얼굴 특징 분석 등의 중간 데이터는 사용자 계정 식별과 연결된 형태로만 앱 서비스 제공 목적에서 한정 사용됨.

4. 제3자 공유 및 책임

- 본 앱은 렌더링된 가상 착용 이미지 생성 시 Google API를 사용하며, 이 때 업로드된 얼굴 사진 및 안경 사진이 API 제공자에게 전달됩니다.
- 외부 API 제공자(Google)는 본 앱과의 계약 및 Google의 개인정보 보호 정책에 따라 해당 데이터 처리(사진 분석, 렌더링)만 수행하며, 본 앱 이외의 다른 목적으로 사용되지 않습니다.
- 데이터 공유 시 전송 암호화 등을 포함한 보안 조치가 적용됩니다.
- 외부 API 제공자(Google)는 전송받은 이미지를 법적/규제적 요구사항 충족 및 위반 여부 확인을 위해 제한된 기간동안 로깅하거나 임시 캐시 형태로 저장할 수 있습니다. 대신 해당 이미지를 데이터 개선 또는 학습 목적으로 사용하지 않으며, 제한된 기간 동안 저장된 데이터 또한 기간 종료 시점에 즉시 삭제합니다.

5. 데이터 보전 및 삭제 정책

- 얼굴 사진, 안경 사진, 그리고 가상 착용 이미지 포함한 관련 업로드된 모든 이미지는 사용자가 해당 이미지 삭제 기능을 통해 삭제 버튼을 클릭할 때 즉시 삭제됩니다.
- 사용자 탈퇴 요청 시, 클라이언트 측 및 서버 측에 저장된 해당 사진 및 관련 가상 착용 이미지와 모든 중간 데이터가 즉시 완전히 삭제됩니다 (로그, 캐시, 백업 저장소 포함).
- 본인 외에는 해당 이미지에 접근할 수 없도록 기술적 조치를 시행하고 있으며, 사용자 식별번호(user ID)와 연결된 얼굴 데이터는 오직 해당 사용자만 볼 수 있는 상태로 관리됩니다.
- 얼굴-안경 사진 및 가상 착용 결과 이미지는 가입 기간 동안 보존되며, 삭제 요청 또는 탈퇴 시 즉시 폐기됩니다.

6. 접근 권한 및 보안 조치

- 이미지 및 가상 착용 결과물은 암호화된 통신(예: HTTPS/TLS)으로 전송되며, 서버 저장 시 암호화 저장됩니다.
- 내부 시스템 및 직원, 운영자 등의 접근은 최소 권한 원칙(least privilege)으로 제한되며, 사용자 본인을 제외한 제3자는 이미지 데이터에 접근할 수 없습니다.

7. 사용자 동의 및 권리

- 앱 가입 시 이용약관 및 개인정보 취급방침에 대해 사용자에게 동의를 구합니다.
- 사진 업로드 시 사용자가 본인의 얼굴 사진 및 안경 사진을 선택해서 업로드 합니다.
- 사용자는 언제든지 앱 내 이미지 삭제 기능을 통해 업로드된 사진/가상 착용 이미지 삭제를 요청할 수 있으며, 삭제는 요청 즉시 이루어집니다.
- 또한 탈퇴 절차를 통해 전체 계정과 연결된 얼굴/안경/결과 이미지 및 관련 데이터가 제거될 수 있습니다.

8. 식별 가능성 및 사용자 식별 번호와의 관계

- 업로드된 얼굴 사진 및 가상 착용 이미지, 안경 사진은 사용자 식별번호(user ID)와 연결되어 관리됩니다.
- 이 연결은 서비스 제공 목적에 한정되며, 사용자 신원(identity)를 공적으로 확인하는 목적으로 사용되거나 외부 공개되지 않습니다.

9. 아동의 개인정보 보호

- 본 앱은 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 법정대리인의 동의 없이는 서비스 이용이 제한됩니다.
